# Envoyer le projet sur GitHub

Le dépôt est configuré en **SSH** : `git@github.com:Chakir24/Blog.git`

## Pousser le code

Ouvre un terminal et exécute :

```bash
cd /home/tatsuya/Desktop/Project/blog
git push -u origin main
```

Assure-toi d’avoir une [clé SSH configurée](https://docs.github.com/fr/authentication/connecting-to-github-with-ssh) sur ton compte GitHub.
